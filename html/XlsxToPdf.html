XlsxToPdf {XlsxToPdf}R Documentation

Transforma um arquivo xlsx em pdf.

Description

Transforma um arquivo xlsx em pdf.

Usage

XlsxToPdf(x, Sheet, Output)

Arguments

x

É o nome e caminho onde está localizado o arquivo xlsx.

Sheet

É o nome da guia no arquivo xlsx que deseja salvar como pdf.

Output

É o nome e caminho onde será salvo o arquivo pdf

Examples

XlsxToPdf("C:/MeuArquivo.xlsx", "Plan1", "C:/MeuArquivo.pdf")
